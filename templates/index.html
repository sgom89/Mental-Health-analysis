Welcome to the Mental Health Data Analysis Tool

This web application allows you to explore and analyze mental health data. Use the navigation menu to access different features:

  • View Data: Examine the first few rows of the mental health dataset
  • Analyze Data: Generate summary statistics and visualizations for specific variables
  • Correlation Analysis: Investigate relationships between different mental health factors
  • Filter Data: Focus on specific subsets of the mental health data

Start your analysis by selecting one of the options from the navigation bar above. This tool aims to provide insights into mental health trends and patterns.
